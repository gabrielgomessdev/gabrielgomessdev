## 📌 Projeto em Destaque

### ReAlimentar | Projeto Acadêmico – PUC Minas

Plataforma web desenvolvida para conectar doadores, instituições e voluntários, promovendo a redistribuição de alimentos excedentes e o combate à insegurança alimentar. 

O projeto foi reconhecido como **Destaque do Eixo 1 na Mostra de Projetos de ADS da PUC Minas** (1º semestre de 2025), recebendo certificado oficial pela relevância da solução e pelo impacto social gerado.

**Contribuições no projeto:**
* Desenvolvimento de funcionalidades de cadastro, autenticação e gerenciamento de doações.
* Implementação do sistema de avaliações de usuários e persistência de dados com `LocalStorage`.
* Execução de testes de software e garantia de boas práticas de desenvolvimento web.
* Trabalho colaborativo ativo utilizando controle de versão com Git e GitHub.

**Tecnologias utilizadas:**
* HTML5, CSS3 e JavaScript
* Git e GitHub (Versionamento e metodologias ágeis)

🔗 [Acesse o projeto na web](https://icei-puc-minas-pmv-ads.github.io/pmv-ads-2025-1-e1-proj-web-t12-pmv-ads-2025-1-e1-proj-realimentar/codigo-fonte/)
